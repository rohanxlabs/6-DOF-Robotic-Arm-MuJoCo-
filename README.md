# 🤖 6-DOF Robotic Arm Simulation using MuJoCo

A high-performance **6 Degrees of Freedom (6-DOF) Robotic Arm** simulation built with **MuJoCo**, designed for robotics research, motion planning, reinforcement learning, and control algorithm development.

This project provides a realistic simulation environment for experimenting with robotic manipulation without requiring physical hardware. MuJoCo is widely used for robotics and machine learning research due to its fast and accurate physics simulation. 0

# ✨ Features

- 6-DOF robotic manipulator
- Realistic physics simulation with MuJoCo
- MJCF robot model
- Joint control
- End-effector visualization
- Easy to modify robot parameters
- Modular project structure
- Suitable for robotics research and education

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/rohanxlabs/6-DOF-Robotic-Arm-MuJoCo-.git
```

Move into the project

```bash
cd 6-DOF-Robotic-Arm-MuJoCo-
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

**Windows**

```bash
.venv\Scripts\activate
```

**Linux / macOS**

```bash
source .venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Simulation

```bash
python simulate.py
```
